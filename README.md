#Text-Mining Core Functionalities
##About
This repository shares ideas, scripts, and small tools for Python-based text mining. The goal is to provide simple and ideas code for working with textual data in different research and practical contexts. In the case we used abstracts and keywords. Some of the scripts here are part of my work on the project related to the "Sustainable Supply Chains.” They cover typical workflows such as preprocessing, vectorization, clustering, topic exploration, and other text‑analytics tasks that support research in logistics and supply chain management.

##JupyterLab Usage
All examples and scripts in this repository were designed to be used in JupyterLab.
